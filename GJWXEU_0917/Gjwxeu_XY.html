SZÓKIRAKÓ JÁTÉK

Adott egy 4*9-es négyzetháló. A program megaf egy - a négyzetháló feletti képhez
kapolódó - szót vagy mondatot, majd ennek betűit összekeverve elhelyezi ebben a
négyzethálóban.

A játékos feladata az, hogy az összekevert betűkből kirakja a szót

vagy mondatot a kijelölt helyen. A betűket vizszintes vagy függőleges irányban egyenként
lehet mozgatni.


Ha a betű útját elállja egy másik betű, akkor az csak a másik betűig tolódik el.

Miután sikerült a betűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely betűket felcseréljük, akkor azokra színkiemeléssel hívja fel a figyelmünket.

Derékszögű háromszögben igaz Pitagorasz tétele:

a2 + b2 = c2

Általános háromszögek esetén is igaz:

α + β + γ = 180°
